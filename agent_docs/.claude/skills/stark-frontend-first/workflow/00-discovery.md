# Workflow 00 — Discovery

Read in order:
1. RECOVERY.md (if exists) at project root
2. `_project/APP_BRIEF.md`
3. `_project/DATA_CONTRACT.md`
4. `_project/UI_SPEC.md`
5. `_project/CLAUDE.md`
6. Project root CLAUDE.md
7. Existing code: `/app`, `/components`, `/services`, `/types`, `/mocks`
8. Starter kit handbook at `agent_docs/starter-kit/`

Verify you can answer:
- What is this app?
- Who uses it?
- What screens does it have in this phase?
- What data shapes flow through it?
- What's already built vs missing?

Surface gaps to operator before proceeding.
